# ATHENA Dotfiles

Personal Linux dotfiles for my Athena OS Hyprland workstation.

> ATHENA MK-I — a custom engineering-focused Linux environment.

![Desktop](screenshots/desktop.png)

## Components

- Hyprland
- Waybar
- Kitty
- Wofi
- Dunst
- SDDM Athena MK-I Theme
- Zsh + Starship
- Power Menu
- Clipboard History
- Screenshot Shortcuts
